# Stocks Analysis

## Quick Start

1.  **Start the Gateway:**
    ```bash
    docker compose up -d
    ```

2.  **Access the GUI (VNC):**
    Open [http://localhost:6080](http://localhost:6080) (Password: `password`)
    *   *Note: You may need to manually accept API connections or configure Trusted IPs here.*

3.  **Run the Client:**
    ```bash
    source .venv/bin/activate
    python connect_ibkr.py
    ```

This is a new repository for the Stocks project.
